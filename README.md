#### Additional information


### pi-topHUB on Raspbian <a name="software-raspbian"></a>
The <b>pi-top</b>HUB software exists on the Raspbian software repositories. Simply run the following commands at the terminal (and then reboot):

```
sudo apt update
sudo apt install pt-hub
```

If you prefer to manually install the packages or want to install a specific set of packages see the [Manual Configuration and Installation](https://github.com/pi-top/pi-topHUB/wiki/Manual-Configuration-and-Installation) page on the wiki.
